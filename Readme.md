# 🧪 Potion Clash - 2D Unity Game

Potion Clash is a 2D adventure game developed in Unity where the player must collect keys, avoid dangerous obstacles, use magical potions, and reach the exit door to complete each level.

## 🎮 Gameplay Objective

The main goal is to explore the level, collect the required key, and open the final door to finish the stage. Along the way, players must avoid hazards, defeat enemies, and strategically use different potions to survive.

## ✨ Features

* Key collection system
* Door unlocking mechanic
* Multiple magical potions with unique abilities
* Enemy encounters
* Health system
* Hazardous environments
* Smooth 2D character movement

## 🧪 Potion Types

### 🟢 Green Potion

Changes the player's movement direction.

### ⚫ Black Potion

Flips the camera view, creating a challenging gameplay experience.

### ❤️ Yellow Potion

Restores or boosts the player's health.

### 🔴 Red Potion

Increases the player's movement speed for a limited time.

## ⚠️ Obstacles

Players must carefully avoid various hazards, including:

* Thorn traps
* Fire-covered ground
* Enemy attacks

Touching these obstacles can reduce health or cause the player to lose the game.

## 👾 Enemies

Enemies patrol the level and create additional challenges. Players must avoid them while collecting keys and reaching the exit.

## 🛠️ Technologies Used

* Unity Engine
* C#
* Unity 2D Physics
* Unity Animator

## 🚀 How to Play

1. Move through the level.
2. Collect the key.
3. Use potions strategically to gain advantages.
4. Avoid thorns, fire, and enemies.
5. Reach the door to complete the level.

## 🚀 Getting Started

### Prerequisites

* Unity Hub
* Unity Editor (recommended version used in the project)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/moaaz-khaled/Potion-Clash-2DGame-Unity.git
```

2. Open Unity Hub.
3. Click **Open Project**.
4. Select the cloned project folder.
5. Allow Unity to import all assets.
